# Movies Info App

A Streamlit web application that fetches detailed information about Bollywood movies, including posters, ratings, box office earnings (converted to INR), and more.  
You can also download the complete movie summary as **CSV** and **PDF**.

---

## Features

- Search any Bollywood movie by name  
- Shows:
  - Movie poster
  - Director and actors
  - Release date
  - IMDb rating (with rating bar)
  - Plot summary
  - Box office (converted USD → INR)
  - Hit or Flop status (based on IMDb rating)
- Saves Last 5 searches in the sidebar
- Download all movie info as **CSV**  
- Download a beautiful **PDF report** including posters and actor list  
- Auto USD → INR conversion using live exchange rate API

---

## Tech Stack

- **Python**
- **Streamlit**
- **OMDB API**
- **ExchangeRate API**
- **FPDF** for generating PDF
- **Pandas**
- **Pillow**
